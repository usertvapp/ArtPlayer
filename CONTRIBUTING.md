This project uses `parcel` as a development and packaging tool, so there are few overall configuration options

## 1. Install dependencies

```bash
npm install
```

or

```bash
yarn
```

## 2. Develop the project and open the demo page

```base
npm run dev
```

or

```bash
yarn dev
```

## 3. Package the project

```bash
npm run build
```

```bash
yarn build
```
